# L'Élégance | Luxury Beauty Salon Landing Page

A sophisticated, single-page landing page designed for high-end beauty salons and spas. This project focuses on **luxury aesthetics**, **smooth user experience**, and **mobile-first responsiveness**.

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Tech](https://img.shields.io/badge/tech-HTML5%20|%20CSS3%20|%20JS-rose)

## 🌸 Aesthetics & Design
- **Color Palette**: Blush White background (#fdf0f3), Deep Rose accents (#b5446e), and Dark Charcoal text (#1e1e1e).
- **Typography**: Elegant *Cormorant Garamond* for headings and clean *Nunito* for body text.
- **Visuals**: Soft radial gradients, floating bokeh animations, and premium AI-generated imagery.

## 🚀 Features
- **Hero Section**: Full-viewport landing with animated floating circles and dual CTA buttons.
- **Interactive Services**: Accordion-style service menu categorized by Hair, Skin, Nails, and Bridal packages.
- **Filterable Gallery**: A 3-column grid with category filters (Hair/Skin/Nails) and elegant hover overlays.
- **Client Testimonials**: Stylish review cards with custom star ratings and initials-based avatars.
- **Location & Hours**: Integrated map layout with a styled schedule table and "Call to Book" functionality.
- **Sticky WhatsApp**: A floating, pulsing WhatsApp button for instant booking.
- **Scroll Animations**: Sections fade in gracefully as the user scrolls using the `Intersection Observer API`.

## 🛠️ Built With
- **HTML5**: Semantic structure for better SEO.
- **CSS3**: Custom animations, transitions, and Flex/Grid layouts.
- **Vanilla JavaScript**: Lightweight logic for filtering, accordions, and scroll reveals.
- **Font Awesome**: Premium iconography.

## 📱 Responsiveness
Fully optimized for all devices:
- Desktop (Ultra-wide supported)
- Tablets (Fluid grid adjustments)
- Mobile (Stacked layouts and minimal navigation)
